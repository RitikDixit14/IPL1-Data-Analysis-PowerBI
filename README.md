# IPL Data Analysis Dashboard (Power BI)

## Overview
Interactive Power BI dashboard analyzing IPL match and ball-by-ball data across multiple seasons.

## Features
- Batting Insights: Top/lowest run scorers, most sixes/fours
- Bowling Insights: Top wicket-takers, economy rates, top fielders
- Team & Toss Analysis: Most successful teams, toss decisions, toss impact on match outcome
- Venue Analysis: Matches per city, lucky stadiums

## Tools Used
- Power BI Desktop
- DAX for custom measures
- Power Query for data cleaning

## Dataset
Source: [Kaggle IPL Dataset](link-if-you-have-it)
